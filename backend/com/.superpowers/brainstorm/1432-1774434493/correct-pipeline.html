每个 Panel（分镜）的生产管线

Panel = 分镜 = 视频生成的最小单位（前端文档中的 Segment = 后端的 Panel）

核心概念

前端 Segment = 后端 Panel，概念统一。一个 Panel 就是一个分镜，是生产的最小单位。

单 Panel 生产流程

BG
生成背景图 自动
根据分镜的 scene_description 调用图像生成 API，生成一张场景背景图
C
匹配角色资产 自动
从 Step4 生成的角色素材中自动匹配该分镜涉及的角色参考图
4G
AI 生成四宫格漫画 审核点
背景图 + 角色参考图 → 调用大模型一键生成四宫格漫画（2×2），带序号标注
V
AI 视频生成 手动触发
四宫格漫画 → 作为参考图上传给视频大模型 → 生成视频片段
场景1 + 角色
场景2 + 角色
场景3 + 角色
场景4 + 角色
四宫格漫画 = AI 一键生成（背景 + 角色 → 融合图，2×2 布局，带序号）
旧流程（废弃）：九宫格参考图 → 前端 Canvas 手动融合 → 上传融合图 → 视频生成
新流程：背景图 + 角色资产 → AI 一键生成四宫格漫画 → 用户审核 → 视频生成
数据模型关系

Chapter → Episode → Panel (分镜)
                           ├─ 1 背景图
                           ├─ N 角色参考图
                           ├─ 1 四宫格漫画（AI 融合，审核点）
                           └─ 1 视频片段

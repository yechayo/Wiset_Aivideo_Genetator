完整生产管线

基于后端 Service 代码梳理的真实流程

Step5 页面 → 全部生产流程（分镜生成 → 场景图 → 融合 → 视频）
Step6 页面 → 视频拼接 + 下载
STEP 5 — 全部生产
1分镜生成 (Storyboard Generation)
LLM 根据剧本内容生成分镜 JSON，包含每格的场景描述、角色状态、镜头参数
StoryboardService → TextGenerationService (LLM API)
POST /panels/generate — AI 生成分镜
POST /panels/confirm — 确认分镜（用户审核后）
POST /panels/revise — 基于反馈修改分镜
POST /panels/retry — 重试失败生成
↓
2分镜增强 (Storyboard Enhancement)
为分镜添加专业镜头参数：景别（大远景→大特写15种）、角度（平视→俯视7种）、运动（固定→环绕12种）
StoryboardEnhancementService → TextGenerationService (LLM API)
（generate 内部自动执行，无需单独接口）
↓
3场景分析 & 分组 (Scene Analysis)
分析分镜内容，按场景/角色/风格分组，决定哪些分镜可以合并处理
SceneAnalysisService — 纯本地逻辑，无外部 API
（produce 内部自动执行，无需单独接口）
↓
4参考网格图生成 (Scene Grid)
为每组分镜生成九宫格参考图（≤6格用2×3，>6格用3×3），每格1024×576
SceneGridGenService → ImageGenerationService (AI 图像 API)
GET /panels/grid-info — 获取网格URL、尺寸、角色参考图
↓
5网格切分 & 融合选择 (Grid Split & Fusion)
后端将九宫格切分为单格图，前端用户在 Canvas 上手动融合角色参考图到场景图上，然后上传融合结果
GridSplitService → OssService (阿里云 OSS)
POST /panels/split-grid-page — 后端切分单页网格
POST /panels/fusion-image — 上传前端 Canvas 导出的融合图
POST /panels/submit-fusion-page — 提交单页融合结果
GET /panels/panel-states — 查看所有格子的融合+视频状态
↓
6单格生产流程 (Per-Panel Pipeline)
每格分镜依次执行：背景生成 → 融合 → 转场 → 视频生成，可逐格操作或批量自动执行
PanelProductionService → ImageGenerationService (背景/融合) → SeedanceVideoService (视频)
GET/POST /panels/{id}/background — 背景生成
GET/POST /panels/{id}/fusion — 角色融合
GET/POST /panels/{id}/transition — 转场生成
GET /panels/{id}/tail-frame — 尾帧信息
GET /panels/{id}/video-task — 视频生成任务状态
POST /panels/{id}/produce — 一键生产单格（背景+融合+转场+视频）
GET /panels/{id}/production-status — 单格完整生产状态
POST /panels/auto-produce-all — 批量自动生产所有分镜
STEP 6 — 视频拼接 & 下载
7字幕生成 (Subtitle)
从分镜对话中提取文本，生成 SRT 字幕文件，与视频时间轴对齐
SubtitleService — 本地逻辑
（合成时自动执行，或提供单独接口）
↓
8视频合成 (Video Composition)
用 FFmpeg 将所有分镜视频片段按顺序拼接为最终视频，可选叠加字幕
VideoCompositionService → FFmpeg (本地)
GET /panels/video-segments — 查看已完成的所有视频片段
POST /panels/synthesize — 合成最终视频
↓
9下载 & 管理
提供最终视频的下载链接和剧集管理
GET /episodes/{id} — 剧集详情（含 finalVideoUrl）
（下载走 OSS 直链，无需额外接口）
外部 API 调用点:
① LLM API — 分镜生成 & 增强（Stage 1-2）
② AI 图像 API — 参考网格图 & 背景生成 & 融合（Stage 4, 6）
③ Seedance/火山引擎 API — 视频生成（Stage 6）
④ 阿里云 OSS — 文件存储（全流程）
⑤ FFmpeg — 本地视频合成（Stage 8）

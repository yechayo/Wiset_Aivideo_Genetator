API 路由设计

ProjectController 只管 CRUD + 状态，ScriptController 管剧本

ProjectController — /api/projects
POST /api/projects 创建项目
GET /api/projects?page&size&status&sortBy 项目列表（分页）
GET /api/projects/{id} 项目详情
PUT /api/projects/{id} 全量更新
PATCH /api/projects/{id} 部分更新
DELETE /api/projects/{id} 逻辑删除
GET /api/projects/{id}/status 状态详情（含导航）
POST /api/projects/{id}/status/advance 推进/回退
ScriptController — /api/projects/{id}/script
GET /script 获取剧本
POST /script/generate 生成大纲
POST /script/confirm 确认剧本（自动批量生成剧集）
POST /script/revise 修改剧本
PATCH /script/outline 手动保存大纲
已移除：generate-episodes、generate-all-episodes（确认时自动批量生成）

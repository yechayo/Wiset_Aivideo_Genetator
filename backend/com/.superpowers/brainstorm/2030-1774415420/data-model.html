数据模型设计

三个实体统一 Map 结构 + Project 新增 deleted 字段

Project 实体
// 独立字段 id: Long (AUTO) projectId: String userId: String status: String deleted: Boolean (默认 false) ← 新增 createdAt: LocalDateTime updatedAt: LocalDateTime // projectInfo Map 的 key 结构 { "storyPrompt": "故事提示词", "genre": "热血玄幻", "targetAudience": "18-30", "totalEpisodes": 10, "episodeDuration": 60, "visualStyle": "ANIME", "script": { "outline": "Markdown 全文（展示用）", "characters": [ { "name": "林墨", "role": "主角", "personality": "...", "appearance": "...", "background": "..." } ], "items": [ { "name": "暗影令牌", "description": "..." } ], "episodes": [ { "ep": 1, "title": "暗夜书店", "synopsis": "...", "characters": ["林墨"], "keyItems": ["暗影令牌"] } ] } }
Episode 实体
// 独立字段 id: Long (AUTO) projectId: String status: String // DRAFT, DONE, FAILED, ... createdAt: LocalDateTime updatedAt: LocalDateTime // episodeInfo Map 的 key 结构 { "episodeNum": 1, "title": "暗夜书店", "content": "完整剧本内容...", "characters": "涉及角色文本", "keyItems": "关键物品文本", "continuityNote": "连贯性备注", "visualStyleNote": "视觉风格备注", "synopsis": "大纲中的梗概（从 script.episodes 复制）", // 分镜阶段由 StoryboardService 填充 "storyboardJson": "...", "errorMsg": null, // 生产阶段由 EpisodeProductionService 填充 "productionStatus": "NOT_STARTED" }
Character 实体
// 独立字段 id: Long (AUTO) projectId: String status: String createdAt: LocalDateTime updatedAt: LocalDateTime // characterInfo Map 的 key 结构 { "charId": "CHAR-xxx", "name": "林墨", "role": "主角", "personality": "冷静沉稳", "appearance": "黑发、深色风衣", "background": "前特工", // 图像阶段填充 "images": { "front": "url", "side": "url", "back": "url" }, "expressionImages": { "happy": "url", "angry": "url", ... } }

Repository 变更

ProjectRepository 所有查询加 deleted = false 条件。可通过 MyBatis-Plus @SqlCondition 注解或全局配置实现。

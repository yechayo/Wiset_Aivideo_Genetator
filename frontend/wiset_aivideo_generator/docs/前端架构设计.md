# Wiset AI视频生成器 - 前端架构设计

## Context

基于现有的 React 19 + TypeScript + Vite + Tauri 技术栈，设计一个完整的 AI 视频生成应用前端架构。项目需要支持：
- 多页面导航（登录、创作、视频管理、个人中心等）
- 复杂的状态管理（用户信息、视频生成任务、编辑状态等）
- 文件上传/下载管理
- 视频预览和编辑
- 桌面端原生功能（通过 Tauri）

---

### 1. 路由架构：React Router v7

**选择理由：**
- React 官方推荐，生态成熟
- 支持嵌套路由、懒加载
- 与 React 19 完美兼容
- TypeScript 支持优秀

**核心依赖：**
```json

  "react-router": "^7.0.0",
  "react-router-dom": "^7.0.0"
}
```

**路由结构设计：**
```

/create                     → 视频创作
```

### 2. 状态管理示例 (Zustand)

```typescript
// stores/authStore.ts
import { create } from 'zustand';

interface AuthState {
  user: User | null;
  token: string | null;
  login: (credentials: LoginRequest) => Promise<void>;
  logout: () => void;
}

export const useAuthStore = create<AuthState>((set) => ({
  user: null,
  token: localStorage.getItem('token'),
  login: async (credentials) => {
    const res = await authService.login(credentials);
    set({ user: res.data.user, token: res.data.token });
  },
  logout: () => {
    set({ user: null, token: null });
    localStorage.removeItem('token');
  },
}));
```

### 3. 路由配置示例

```typescript
// router/index.tsx
import { createBrowserRouter, Navigate } from 'react-router-dom';
import { AppLayout } from '@/components/layout/AppLayout';
import { LoginPage } from '@/pages/auth/LoginPage';
import { DashboardPage } from '@/pages/dashboard/DashboardPage';

const router = createBrowserRouter([
  {
    path: '/',
    element: <AppLayout />,
    children: [
      { index: true, element: <Navigate to="/dashboard" replace /> },
      { path: 'dashboard', element: <DashboardPage /> },
      { path: 'create/*', element: <CreateRoutes /> },
      { path: 'videos/*', element: <VideoRoutes /> },
      { path: 'profile', element: <ProfilePage /> },
    ],
  },
  { path: '/login', element: <LoginPage /> },
]);

export default router;
```

### 4. 路由守卫

```typescript
// router/guards.tsx
import { Navigate } from 'react-router-dom';
import { useAuthStore } from '@/stores';

export const ProtectedRoute = ({ children }) => {
  const token = useAuthStore((state) => state.token);
  return token ? children : <Navigate to="/login" replace />;
};
```
